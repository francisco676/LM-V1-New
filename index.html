The 5 Agents (and What They Handle)

Each agent is a Cowork skill that runs inside the Claude desktop app. No code. No complex setup. Install the skill, describe your task, let it run.

📋
Agent 1 — Meeting Prep
Replaces: $40–60k/yr EA or Chief of Staff

Before every sales call or investor meeting, this agent scans your Gmail, Slack, Notion, and calendar to build a 1-page brief: who they are, what you've discussed before, open action items, and suggested talking points.

Saves 45–60 minutes of research before every important call.
✍️
Agent 2 — LinkedIn Content Engine
Replaces: $3–5k/mo content agency

Every week: finds 5 trending topics across your niches, writes 5 LinkedIn posts in your exact voice, creates 5 lead magnet files (PDF or Word docs), and uploads everything to a new Notion page — ready to schedule.

12+ LinkedIn post ideas generated in the first 14 days.
🧲
Agent 3 — Lead Magnet Machine
Replaces: $2–4k/mo freelance designer/copywriter

Turns any LinkedIn post into a gated landing page — dark-branded, Beehiiv-connected, Vercel-ready. Subscribers unlock the full playbook. Every subscriber goes straight into your newsletter list. Zero manual work.

Full HTML landing page generated and deployed in under 10 minutes.
📁
Agent 4 — File & Machine Organizer
Replaces: 2–3 hours of manual cleanup/week

Sorts your Downloads folder, Desktop, and Screenshots using GTD principles — automatically buckets everything into Action, Reference, Archive, or Trash. Weekly review takes 5 minutes instead of 45.

Desktop went from 140 files to 12 in the first run.
📊
Agent 5 — Investor Update Writer
Replaces: 3–4 hours of monthly writing

Pulls your metrics from Notion, Gmail threads, and Slack, then writes a complete monthly investor update in your voice — highlights, lowlights, key metrics table, ask section, and next-month focus. One prompt. Done.

Monthly update written and ready to send in under 8 minutes.

How to Set This Up in 20 Minutes

You don't need to know how to code. If you can install an app and type a sentence, you can deploy all 5 agents today.

1

Download Claude Desktop + Enable Cowork Mode

Go to claude.ai, download the desktop app, and enable Cowork mode from the settings panel. This activates the agent/skill layer that all 5 agents run on.

2

Install the Plugin Pack

In Cowork settings, install the RallyUp plugin pack. This drops all 5 agent skills into your sidebar in one click — Meeting Prep, Content Engine, Lead Magnet Machine, File Organizer, and Investor Update.

3

Connect Your Apps (5 min)

Connect Gmail, Slack, Notion, and Google Calendar via the MCP connectors in Cowork settings. Each is a one-click OAuth — no API keys, no dev setup. This is what lets the agents pull real context from your actual work.

4

Run Your First Agent

Type "prep me for my meeting with [name]" or "run my weekly content" in the Cowork chat. The agent activates, pulls live context, and delivers the output. That's it.
