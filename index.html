The Agents

The full ops stack.

Agent 01

The Meeting Prep Agent

Trigger: 2 hours before any calendar event

Before every meeting, this agent reads your calendar invite, pulls recent emails from that contact, checks your last Notion notes from meetings with them, and surfaces the 3 most important things to know. You walk in prepared every time — zero effort.

Starter prompt
I have a meeting with [name] at [time]. Pull their last 3 emails from my Gmail. Check my Notion for any previous meeting notes with them. Look up their LinkedIn and summarize what they do in 2 sentences. Then give me:1. The 3 most important things to know before this meeting 2. 2 questions I should ask 3. Any open action items from last time
→Time saved: 20–30 min of pre-meeting research, every meeting
→Works with: Gmail, Notion, Google Calendar (all via MCP connectors)
Agent 02

The Content Engine

Trigger: Weekly, Sunday morning

Searches for what's trending in your niche, drafts 5 LinkedIn posts in your voice, creates gated lead magnets for each one, and uploads everything to a Notion content calendar. The whole run takes under 3 minutes of wall-clock time. You show up Monday with a full week of content ready.

Starter prompt
Run my weekly LinkedIn content workflow. 1. Search for 5 trending topics in: AI tools, employee advocacy, LinkedIn growth 2. Draft 5 posts in my voice (casual, lowercase, hook-first) 3. Create a lead magnet PDF or HTML for each post 4. Upload everything to my Notion content calendar Today is [date]. Schedule posts starting Monday.
→Output: 5 posts + 5 lead magnets + Notion page, every week
→Replaces: 4–6 hours of content creation per week
Agent 03

The Lead Magnet Machine

Trigger: After drafting any LinkedIn post

Takes a LinkedIn post and builds a full gated HTML lead magnet around it — email gate, branded design, Beehiiv integration, RallyUp CTA, and all content. Deployed to Vercel as a static site. The whole thing, start to finish, in one command.

Starter prompt
Create a gated HTML lead magnet for this LinkedIn post: [paste your post here] Requirements: - Email gate using Beehiiv (pub ID: pub_78884462-...) - RallyUp brand colors (blue #5B6FFF, lime #D8EC6A) - 5–7 sections of real, actionable content - Book a Demo CTA: https://calendar.app.google/SDA5TMRMbqbPQWEn7 - Save to /outputs/[topic-name]/index.html
→Output: Deployable HTML file, ready for Vercel in under 2 min
→Captures: Email subscribers + routes hot leads to book a demo
Agent 04

The File Organizer

Trigger: Weekly, Friday EOD

Scans your Downloads and Desktop, GTD-sorts everything into Action / Reference / Archive / Trash buckets, and moves files to the right folders without asking. Screenshots get renamed and filed. Old files get flagged for deletion. Your machine stays clean automatically.

Starter prompt
Run my weekly file cleanup. Scan: ~/Downloads and ~/DesktopSort everything into GTD buckets: - Action: things I need to act on - Reference: things I'll want to find later - Archive: things to keep but not act on - Trash: things that can go Move files to the right folders. Rename screenshots with descriptive names. Flag anything older than 30 days I haven't opened.
→Time saved: 20–40 min of manual file cleanup per week
→Result: Clean machine, every Friday, zero effort
Agent 05

The Investor Update Writer

Trigger: Monthly, first Monday

Pulls MRR and transaction data from Mercury, checks Notion for product updates and milestones, reads your last investor email for tone reference, and drafts the full monthly update. You spend 5 minutes editing instead of 90 minutes writing from scratch.

Starter prompt
Draft my monthly investor update for [month]. Pull from: - Mercury: MRR, transaction volume, burn rate - Notion: product milestones, team updates - Gmail: last investor update for tone reference Format: - Subject: [Company] Update — [Month] [Year] - 3-bullet executive summary - Metrics table (MRR, growth %, key stats) - 3 wins, 2 challenges, top ask - Keep it under 400 words
→Time saved: 60–90 min of writing and data-gathering, every month
→Works with: Mercury, Notion, Gmail (all via MCP connectors)

Getting Started

How to set up your first agent.

Each agent takes about 10 minutes to set up. Here's the sequence that works best:

  • Install Claude Cowork — the desktop app, not just the API. Agents need file system access to be useful.
  • Connect your tools — Gmail, Notion, Google Calendar, Mercury. Each takes ~2 min with OAuth. The connectors unlock 80% of what makes agents actually useful.
  • Start with the Meeting Prep Agent — it's the easiest to set up and the one you'll feel immediately. One prep run pays for itself.
  • Build a skill for your weekly content run — copy the Content Engine prompt above, save it as a scheduled skill, and run it every Sunday. The ROI here is massive.
  • Add the File Organizer last — it needs file system access to your computer. Once you've tested agents that don't touch your files, you'll trust this one more.
